# Intelligent Livestock Health and Behavior Management System

## 📌 Overview

This project is an IoT + Machine Learning-based system designed to monitor livestock health and predict diseases in real time.

## 🚀 Features

* Real-time livestock monitoring
* Health prediction using Machine Learning
* Anomaly detection
* Web dashboard using Flask
* Data visualization and analysis

## 🧠 Technologies Used

* Python
* Flask
* MySQL
* Pandas
* Scikit-learn
* IoT Sensors (Simulated Data)

## 📊 Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest (Best Model)

## ⚙️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
python app.py
```

3. Open browser:

```
http://127.0.0.1:5000
```

## 📁 Dataset

* animal_health_clean.csv

## 🎯 Objective

To develop a real-time intelligent system for livestock health monitoring and early disease prediction.

## 📌 Output

* Health prediction (Healthy / Unhealthy)
* Disease identification
* Dashboard visualization
